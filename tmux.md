# tmux チートシート

## セッション開始
```
# 新規セッション開始
tmux

# 名前をつけて新規セッション開始
tmux new -s <セッション名>
```
## セッション終了
```
# セッションを終了 
※-t <対象セッション名>でセッション名の指定も可能
tmux kill-session

# tmux全体を終了
tmux kill-server
```
## Ctrl+b -> ? のコマンド
```
%     左右にペイン分割
"     上下にペイン分割
[     スクロール
s      セッションの一覧選択
d      セッションから離脱(デタッチ)
$      セッションの名前変更
ctrl+Z tmuxを一時中断 ※fgで復帰
c      新規ウインドウ作成
w      ウインドウの一覧選択
0-9    指定番号のウインドウへ移動
&      ウインドウの破棄
n      次のウインドウへ移動
p      前のウインドウへ移動
l      以前のウインドウへ移動
'      入力番号のウインドウへ移動
.      入力番号にウインドウ番号を変更
,      ウインドウの名前変更
f      ウインドウの検索

q            ペイン番号を表示
カーソル      指定方向のペインへ移動
!            ペインを解除してウインドウ化
x            ペインの破棄
o            ペインを順に移動
;            以前のペインへ移動
z            現在のペインを最大化/復帰
スペース      レイアウトを変更
Alt-1-5      レイウトを変更
{            ペインの入れ替え(前方向)
}            ペインの入れ替え(後方向)
ctrl+o       ペインの入れ替え(全体
```
